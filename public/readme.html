Wallet Dashboard API - Developer Guide

Overview

This API provides wallet data from multiple blockchains, integrates with Coinbase, and supports NFT data retrieval.

Quick Start

# Install dependencies
npm install

# Set environment variables
cp .env.example .env
# Edit .env with your API keys

# Start the server
npm start

API Keys Required

  • Moralis API Key - Required for blockchain data
  • GoldRush API Key - Optional for DeFi position data
  • Coinbase OAuth Credentials - Optional for Coinbase integration

Tip: If you're seeing 401 errors, make sure your Moralis API key is valid and has not expired.

Common Issues & Solutions

Error: Request failed with status code 400

This typically happens when:

  • The wallet address format is invalid
  • The chain parameter is not supported
  • The wallet has too many transactions for the default limit

Solution: Use the /api/test endpoint to test your parameters one by one.

Error: Request failed with status code 429

You've exceeded the API rate limits.

Solution: Implement retry logic with exponential backoff.

Debugging Tips

  • Use the /api/status endpoint to check API dependencies
  • Check the server logs for detailed error messages
  • Try smaller networks first (e.g., 'optimism' or 'base' instead of 'ethereum')
