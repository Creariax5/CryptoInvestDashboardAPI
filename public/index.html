Wallet Dashboard API

This API provides access to blockchain wallet data across multiple networks.

Developer Guide: For more detailed information, check out the Developer Guide.

Test Endpoints

GET /api/test

Test endpoint to verify API functionality.

Parameters:

  • address - Wallet address (required)
  • chain - Blockchain network (default: eth)

Example: Test Ethereum Wallet

Dashboard Endpoints

GET /api/dashboard

Get comprehensive wallet dashboard data.

Parameters:

  • address - Wallet address (required)
  • networks - Comma-separated list of networks (default: ethereum,polygon,bsc,optimism,arbitrum)

Example: Full Dashboard

NFT Endpoints

GET /api/nfts

Get NFTs owned by a wallet address.

Parameters:

  • address - Wallet address (required)
  • chain - Blockchain network (default: eth)
  • limit - Number of NFTs to return (default: 20)

Example: Get NFTs

GET /api/nfts/:contractAddress/:tokenId

Get details for a specific NFT.

Parameters:

  • contractAddress - NFT contract address (in URL)
  • tokenId - NFT token ID (in URL)
  • chain - Blockchain network (default: eth)

Example: Get NFT Details

GoldRush (formerly Covalent) Endpoints

GET /api/goldrush/balances

Get token balances from GoldRush API.

Parameters:

  • address - Wallet address (required)
  • chain - Blockchain network (default: eth-mainnet)

GET /api/goldrush/protocols

Get DeFi protocol positions from GoldRush API.

Parameters:

  • address - Wallet address (required)
  • chain - Blockchain network (default: eth-mainnet)

Coinbase Integration

GET /api/coinbase/auth

Authenticate with Coinbase.

Example: Connect Coinbase

GET /api/coinbase/data

Get Coinbase account data (requires authentication).

API Status

GET /api/status

Check the health and status of API dependencies.

Example: Check API Status
